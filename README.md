# О тг коннекторе
тг бот на python единственная задача которого получать от телеграма сообщения от пользователя и отправлять их в основное приложение,
согласно [api](https://ashflaare.github.io/bot_documentation/). Плюс он может получать от основного приложения что-то и отправлять в тг.
**функционал:**
- отправка обычных тектовых сообщений (работает в обе стороны)
- отправка клавиатур (работало до переписи API, сейчас не проверял)
- отправка файлов и картинок (работало до переписи API, сейчас не проверял)
# О web заглушке
работает на nodejs, вроде для визуала использовался bootstrap.
основная задача эмулировать основное приложение.
работает максимально топорно, нет разделения пользователей на разные чат, есть только 1 чат, в который все идет. (для заглушки пока достаточно)
